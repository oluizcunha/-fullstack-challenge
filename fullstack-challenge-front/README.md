
### FrontEnd
Requerido [Node.js](https://nodejs.org/) v14 para rodar.
1) Entre na pasta do front-end dentro do repositorio
```sh
cd fullstack-challenge-front
```

2) Instale todas as dependencias do projeto
```sh
npm install
```
3) Inicie o projeto
```sh
npm start
```
